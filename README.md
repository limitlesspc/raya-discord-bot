## Install

First you will need to install the following:

- [Node.js](https://nodejs.org/en/)
- [NPM](https://www.npmjs.com/) comes with Node.js
- [PNPM](https://pnpm.io/)

```
pnpm run setup
```

## Usage

Run the bot:

```
pnpm start
```

Linting:

```
pnpm run lint
```
